What is Palindrome Number ?

=>  It's a number if u read from right to left or from left to right(or reverse direction) it's same .

=>  Like 12321

=>  If u read from right to left and left to right it's same hence it's a palindrome

=>  Now the other number like 12345 if u read from left to right it is 12345 but if u read from right to left it's 54321 hence its not palindrome.

=>  Now Palindrome is a number or it can be word also which remains same if we go forward or backward .
